# Changelog

## v0.1.0 — 2026-06-02

Initial release of **seven-showdoc** — a Bash script that auto-generates ShowDoc API documentation from code comments.

### Features
- Recursively scan source directories for ShowDoc-compliant block comments (`/** ... */`)
- Smart file filtering (text only, under 1MB)
- Cross-platform: Linux, macOS, Windows (Git Bash / MINGW / Cygwin)
